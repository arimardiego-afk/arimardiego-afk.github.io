Escolha como instalar

🤖

Celular ou tablet Android — aplicativo (.apk)

Baixe e instale direto no aparelho, como um app comum. É a forma recomendada no Android.

  1. Toque em Baixar o aplicativo aqui embaixo.
  2. Abra o arquivo baixado (na barra de notificações ou na pasta Downloads).
  3. Se o Android perguntar sobre "instalar apps desconhecidos", permita — esse aviso é padrão para qualquer app instalado fora das lojas, e este vem direto do site oficial do desenvolvedor.
  4. Pronto! O ícone Aulas aparece na tela inicial.

iPhone, iPad, computador e outros — no navegador

Abra o app direto no navegador — não precisa instalar nada. Por enquanto, esta é a única opção para aparelhos que não usam Android (iPhone/iPad, Windows, Mac, Chromebook), pois eles não aceitam arquivos .apk.

  1. Toque em Abrir o app no navegador.
  2. Use normalmente — tudo o que você criar fica salvo no aparelho (veja "Onde ficam os seus dados", abaixo).
  3. Crie o ícone na tela do aparelho para abrir com um toque, como um app de verdade (passo 2, abaixo).

Passo 2

Crie o ícone na tela do seu aparelho

Para a versão do navegador: o app ganha ícone próprio e abre em tela cheia, sem a barra do navegador.

🍎

iPhone e iPad (Safari)

  1. Abra o app no Safari.
  2. Toque no botão Compartilhar (quadrado com seta para cima, na barra de baixo).
  3. Toque em "Adicionar à Tela de Início" e confirme.
🤖

Android (Chrome)

  1. Abra o app no Chrome.
  2. Toque no menu ⋮ (canto de cima, à direita).
  3. Toque em "Adicionar à tela inicial" (ou "Instalar app") e confirme.
💻

Computador (Chrome ou Edge)

  1. Abra o app no navegador.
  2. Clique no ícone de instalar no fim da barra de endereço (um monitor com uma seta para baixo).
  3. Ou: menu ⋮/… → "Instalar Organizador de Aulas". O atalho aparece na área de trabalho e no menu Iniciar.
Importante

Onde ficam os seus dados (versão do navegador)

✓ Sim, tudo fica salvo entre um uso e outro. Você não precisa "salvar" nada — cada alteração é guardada na hora, automaticamente.

📍 Os dados ficam no armazenamento interno do navegador, dentro do seu próprio aparelho — no espaço que o navegador reserva para o site do app. Nada é enviado para a internet: nem para nós, nem para terceiros.

⚠️ Atenção — a única forma de perder os dados

Se você limpar os "dados de navegação" / "dados de sites" do navegador (ou desinstalar o navegador), os dados do app são apagados junto. Por isso, crie o hábito do backup: no app, toque no menu ☰ → Arquivo → Salvar backup. Ele gera um arquivo .json na sua pasta de Downloads — guarde uma cópia no Google Drive, num e-mail para você mesmo ou num pendrive. Para recuperar (ou levar tudo para outro aparelho), use ☰ → Arquivo → Abrir backup.

Limitações da versão no navegador

  • Os dados valem para aquele navegador, naquele aparelho. Abrir o app em outro navegador ou aparelho começa vazio — transfira com o backup .json (menu ☰).
  • Limpar os dados do navegador apaga o que foi criado (veja o aviso acima).
  • O espaço para anexos (PDFs, fotos, áudios) depende do navegador — em geral são vários gigabytes, mais que suficiente para uso escolar.
  • Fora isso, é o mesmo app: funciona offline depois da primeira visita, tem os 5 modos de cor, os 5 idiomas, exporta relatórios, resumos e backups.
